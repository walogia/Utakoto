# Utakoto
从View窗口选择获取模型信息插件，方便查看模型参数，凡是label都可点击复制到剪贴板，到对应label粘贴即可，方便改参数

<br/>
**本插件结构借鉴了 韦驮天 开发的 PEBrowser插件**
（项目：https://github.com/weituotian/PEBrowser）

<br/>
说实话，本插件没什么乱用，只是用来学习研究别人的模型参数的（为未来开发自动化插件积累数据和经验）
